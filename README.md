# airlab2 핀볼 게임

간단한 HTML5 캔버스 기반 핀볼 게임 예제입니다. 저장소를 클론한 뒤 `game/index.html` 파일을 브라우저에서 열어 바로 실행할 수 있습니다.

## 실행 방법

1. 저장소를 클론하거나 다운로드합니다.
2. `game/index.html` 파일을 더블 클릭하거나, 웹 서버를 통해 서빙합니다.
3. 게임 화면이 나타나면 다음 조작으로 플레이할 수 있습니다.
   - `←` / `→` : 좌·우 플리퍼 조작
   - `Space` : 게임 오버 이후 다시 시작

## 게임 요소

- 중력과 벽에 반응하는 공 1개와 3개의 라이프
- 좌/우 플리퍼, 레일, 범퍼, 작은 핀(peg) 오브젝트
- 범퍼 또는 플리퍼에 부딪힐 때마다 점수가 증가하며, 남은 공 수와 함께 HUD에 표시됩니다.

즐겁게 플레이해 보세요!
